# Jornadas Web

Aplicación para dividir pagos y seleccionar pagador aleatorio.

## Características
- Calculadora de división de pagos
- Selector aleatorio con animaciones
- Interfaz responsiva
- Efectos visuales

## Instalación
1. Clona el repositorio
2. Abre index.html en el navegador

## Tecnologías
- HTML5
- CSS3
- JavaScript
